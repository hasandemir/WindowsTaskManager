        Automation rules

        If a process matching match has metric op threshold for for seconds, run action.

        OnNameMatchMetricOpThresholdFor (s)ActionCooldown (s)
        Help & examples
        • metric=memory_bytes op=>= threshold=4000000000 for=30 — kill if process uses ≥ 4 GB for 30s
        • metric=cpu_percent op=>= threshold=90 for=60 action=suspend — suspend runaway at 90% for 60s
        • metric=thread_count op=>= threshold=500 — alert on thread leaks
        • Match is a case-insensitive substring of the process name (e.g. chrome.exe, node).
        • Cooldown defaults to 60s — prevents repeated kills of restarted processes.
